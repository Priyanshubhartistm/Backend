How to connect MongoDB Compass:-
1. go to terminal
2. MongoDB Service Start Karein 
sudo systemctl start mongod
3.Status Check Karein
sudo systemctl status mongod


How to Connect to ClusterOne:-
1.mongodb atlas login
2. click on create ClusterOne
3. chosse free version and write name.
4. then fill the section of connect cluster
5. copy the connection string + with your own password.
6. then go to mongodb compass , click on add new connection(paste this link with pass)
7. then save and click.
